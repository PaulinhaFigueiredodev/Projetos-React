# Repositório de Desafios React

Este repositório foi criado para organizar e documentar os desafios de estudo em **React**.

Cada desafio ficará em uma pasta própria, identificada pelo número e pelo nome do desafio. Dentro de cada pasta, haverá um arquivo `README.md` explicando o objetivo do desafio, os requisitos e o resultado esperado.

A ideia é manter os desafios bem organizados, facilitando o acompanhamento da evolução dos estudos e a consulta dos projetos desenvolvidos.
